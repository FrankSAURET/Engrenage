# Engrenage
-----
Version 2026.1
For inkscape V 1.3 +
-----

Extension pour Inkscape permettant de tracer des engrenages à développante et des engrenages à pas métrique (poulie crantée) ; permet d'ajouter des branches, une croix centrale, le module métrique.

À l'origine, copie de inkscape-gears-dev (https://github.com/jnweiger/inkscape-gears-dev).

Utilisation
-----
Copiez le dossier **engrenage** dans le répertoire des extensions d'Inkscape (sous Windows : **%appdata%\inkscape\extensions**).
Si différent, le répertoire peut être trouvé dans Préférences/Système.

Le fichier ini peut être modifié pour ajouter une forme de trou. Si vous faites cela, vous devez ajouter le nom dans la liste déroulante du fichier inx (après la ligne 62). Le nom dans l'attribut value doit être identique à celui du fichier ini (nom entre crochets []).

Redémarrez Inkscape.

L'extension sera disponible dans le menu « Extensions > Formes > Engrenage... ».

Elle peut être installée et mise à jour avec l'extension Màj : **extensions > Mise à jour des extensions de Frank SAURET...**.

Màj est téléchargeable  ici : https://github.com/FrankSAURET/Maj
